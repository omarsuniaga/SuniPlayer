# Skill: Code Reviewer
Description: Ensures code quality and security standards.

## Responsibilities
- Check for anti-patterns and performance bottlenecks.
- Validate that types are correctly used.
- Ensure no sensitive information is leaked.
- Verify that code logic matches the user requirements.

## Protocol
- Must review every major component migration.
- Must flag mismatches between source-of-truth docs and implemented code.
