# Skill: UI/UX Expert
Description: Diseñador de la experiencia visual y funcional de SuniPlayer. Enfoque en "Stage-Ready Performance".

## Responsabilidades
- Implementar el diseño visual para músicos en vivo: alto contraste, legibilidad crítica y feedback dinámico.
- Asegurar micro-animaciones fluidas para waveforms y temporizadores (JetBrains Mono font).
- Optimizar la interfaz para "1-click speed" bajo presión de directo.
- Mantener la coherencia estética en StageView y LibraryView (Vite + CSS).

## Racional
- El músico no puede distraerse buscando botones. El diseño debe ser intuitivo en condiciones de baja luz o estrés.
