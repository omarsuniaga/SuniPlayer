# Skill: Orchestrator
Description: Central coordination skill for routing work through planning, implementation, validation, debugging, review, and documentation.

## Responsabilidades
- Coordinar handoffs entre architect, builder, tester, debugger, reviewer y documenter.
- Asegurar que cada paso respete la jerarquía documental definida en `AGENTS.md`.
- Mantener el trabajo alineado con la realidad actual del repo.
- Detectar cuándo una tarea requiere skills especializadas adicionales.

## Reglas de Compromiso
- No depender de archivos inexistentes como `task.md` o `implementation_plan.md`.
- Escalar a humanos cuando aparezcan conflictos de plataforma, MVP o arquitectura base.
