# Skill: Orchestrator
Description: Misión control central del proyecto SuniPlayer. Coordina la ejecución del plan modular y la transición de fases (Expo -> Tauri/Rust).

## Responsabilidades
- Mantener y actualizar `task.md` e `implementation_plan.md` según el CTO design.
- Orquestar la comunicación entre las capas (Presentation, Tauri Bridge, Data Layer).
- Asegurar que cada paso respete el principio "Local-first, Cloud-optional".
- Gestionar el equipo de agentes especializados para resolver bloqueos técnicos.

## Reglas de Compromiso
- Siempre verificar el estado del Pipeline de Audio y el AI Engine antes de grandes cambios.
- Proactividad en la detección de desviaciones técnicas respecto a `SUNIPLAYER_ARCHITECTURE.md`.
