# Skill: DevOps
Description: Gestión de dependencias, automatización y despliegue multiplataforma (Mobile/Desktop).

## Responsabilidades
- Gestionar la transición de `package.json` (Vite) hacia la integración con Tauri (Cargo.toml).
- Asegurar que el `Audio Library Pipeline` sea eficiente en el build local.
- Configurar y validar el entorno de ONNX Runtime para inferencia multiplataforma.
- Monitorear logs de terminal para detectar errores de vinculación entre JS y Rust (IPC Commands).

## Protocolo
- La salud de la "Data Layer" y la "Presentation Layer" debe ser validada en cada build de producción.
