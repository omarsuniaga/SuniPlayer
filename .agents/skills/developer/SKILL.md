# Skill: Developer
Description: Motor de ejecución para el desarrollo de features alineado con el Stack Tecnológico definido.

## Responsabilidades
- Implementar lógica de frontend usando React/Vite + Zustand.
- Asegurar que el código sea "Local-First" y maneje correctamente la latencia en el renderizado de waveforms.
- Integrar servicios de IA (BPM Detector, Mood Classifier) en la lógica de negocio.
- Resolver tickets técnicos garantizando que el código sea portable a Tauri (Rust Bridge).

## Guía de Estilo
- Seguir el principio de "Separation of Concerns": Estado en `store/`, Lógica en `services/`, Visuals en `components/`.
