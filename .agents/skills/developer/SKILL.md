# Skill: Developer
Description: Execution skill for implementing changes in the active SuniPlayer codebase.

## Responsabilidades
- Implementar lógica de frontend usando React/Vite + Zustand.
- Mantener separación clara entre UI, store, services y data.
- Respetar contratos actuales como `duration_ms` y las validaciones disponibles.
- Resolver tickets con el menor alcance correcto.

## Skills complementarias
- `component-factory` para UI compuesta
- `design-system` y `ui-ux` para consistencia visual
- `systematic-debugging` si la implementación falla en validación

## Guía de Estilo
- Seguir el principio de "Separation of Concerns": Estado en `store/`, Lógica en `services/`, Visuals en `components/`.
