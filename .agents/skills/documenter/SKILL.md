# Skill: Documenter
Description: Keeps the project readable for humans and agents.

## Responsibilities
- Create and update README.md files.
- Write Walkthroughs for major changes.
- Ensure JSDoc/TSDoc comments are informative.
- Maintain the project wiki or documentation artifacts.
- Keep `DECISIONS.md`, `ROADMAP.md`, `TASKS.md`, and `TESTING.md` aligned with actual code changes.

## Companion skill
- `knowledge-base`
