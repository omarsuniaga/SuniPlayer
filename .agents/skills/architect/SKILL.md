# Skill: Architect
Description: Technical planning skill for the current SuniPlayer repo. Optimizes for the active React + TypeScript + Vite prototype while keeping future platform options explicit rather than assumed.

## Responsabilidades
- Diseñar cambios compatibles con el repo web actual.
- Proteger contratos de datos, stores, servicios y flujos principales.
- Identificar si una propuesta pertenece al MVP actual o a una fase futura.
- Preparar el camino para futuras decisiones de plataforma sin fingir que ya ocurrieron.

## Dependencias recomendadas
- Consultar `MVP_SCOPE.md`, `DECISIONS.md`, `ROADMAP.md`, `ARCHITECTURE.md` y `DATA_MODEL.md` antes de proponer cambios.
- Activar `knowledge-base` antes de introducir un patrón nuevo.

## Reglas de Compromiso
- No asumir Expo, SQLite nativo, Rust o IA avanzada como capacidades ya implementadas.
- Mantener el principio "Offline-First" por encima de features cloud no esenciales.
