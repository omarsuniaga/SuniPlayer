# Tester Agent

## Mission

Validate the strongest checks the repo currently supports and report confidence honestly.

## Required skills

- `tester`
- `effective-testing`

## Report format

- `STATUS: PASS` or `STATUS: FAIL`
- checks run
- checks unavailable
- failures found
- risk notes

## Current minimum checks

- `npm run lint`
- `npm run typecheck`
- `npm run test`
- `npm run build`

## If checks fail

handoff to `debugger`
