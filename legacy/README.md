# Legacy Assets

This directory stores reference material and superseded prototypes that are no longer part of the active application entrypoint.

- `web-prototype/SuniPlayer_legacy.jsx` keeps the original single-file prototype for historical reference.
- The active app entrypoint is `src/main.tsx`, which mounts `src/App.tsx`.

Legacy files should not be treated as the source of truth for current architecture or data contracts.
