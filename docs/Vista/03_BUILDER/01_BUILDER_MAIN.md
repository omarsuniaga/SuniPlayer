# VISTA: GENERADOR (Builder Engine)

## Propósito
Motor estratégico para la creación automatizada de sets musicales coherentes y profesionales. Es el cerebro de planificación de SuniPlayer.

## 🏗️ Interfaz Cockpit (v2.0)
- **Diseño de Alta Densidad**: Todo el control estratégico a la vista.
- **Panel de Estrategia Colapsable**: Ajustes de tiempo, energía y reglas DJ sin scroll lateral.
- **Visualizador de Salud de Mezcla**: Indicadores técnicos entre cada track del set generado.

## 🛠️ Inteligencia DJ (DJ Brain)
1. **Anclajes (Pins 📌)**: Permite fijar tracks en posiciones exactas. El algoritmo de Monte Carlo reconstruye el set uniendo los puntos fijos de forma armónica.
2. **Smart Replace (Swap ✨)**: Botón para reemplazar un track individual. Busca en el repertorio el mejor match de BPM y Tono para ese hueco específico.
3. **Mezcla Armónica**: Flag para obligar al algoritmo a seguir el Círculo de Quintas (Camelot).
4. **Energy Flow**: Control de narrativa energética basado en Moods.

## 🧪 Simulador de Set
- **Análisis Predictivo**: Muestra el puntaje armónico total y la duración exacta predicha del show.
- **Validación Stage-Ready**: Chequeo de transiciones antes de cargar al reproductor.

*El Generador convierte una biblioteca en una narrativa musical autónoma.*
